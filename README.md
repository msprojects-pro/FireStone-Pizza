Firestone Pizza | Artisanal Wood-Fired Kitchen

Welcome to the official repository for Firestone Pizza. This project is a premium digital experience designed for a high-end pizzeria, showcasing artisanal craftsmanship, fresh ingredients, and a modern dining atmosphere.

✨ Key Features
Rustic-Modern Design: An atmospheric visual style using warm earth tones and high-resolution food imagery.
Interactive Menu: A seamless, responsive layout showcasing gourmet pizza selections and seasonal sides.
Optimized Performance: Built with Vite for instant loading and smooth scrolling across all devices.
Seamless UI/UX: Designed with a conversion-focused layout to encourage online browsing and reservations.
